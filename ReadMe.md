# このソフトウェアについて

tabulator.jsでTSVからテーブルを作成する。

# 開発環境

* Linux Mint 17.3 MATE
* Firefox 42.0
* [d3.js](https://d3js.org/)
* [jQuery.js](https://jquery.com/)
* [tabulator.js](https://github.com/olifolkerd/tabulator)
